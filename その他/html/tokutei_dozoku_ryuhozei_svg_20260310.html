まず結論

特定同族会社の留保金課税は、「特定同族会社か」 と 「留保金額が留保控除額を超えるか」 の二段階で動く。前者は別表二で判定し、後者は別表三(一)で 欄9 → 欄19 → 欄20 → 欄21 → 欄1〜8 → B01欄9外書 と流れる。したがって、実務上の核心は、B04の留保列を正しく欄9へ拾うこと、法67条4項の「期末配当等」を欄11へ正しく入れること、そして付表一で留保控除額の最大値を選ぶことである。

さらに、法67条3項は、受取配当益金不算入や繰越欠損金の損金算入などを織り込んだ「所得等の金額」を土台にしており、国税庁手引も、特別控除等があると会計上の所得が0でも留保金課税が残りうると注意喚起している。つまり、留保金課税は単なる「利益剰余金が多い会社への上乗せ税」ではなく、社内に残った利益を税務用に再構成し直してから累進税率を掛ける制度である。

1. 詳細SVG

中央の本流が B03(一) の作成順序、左列が転記元の別表、右列が税率と落とし穴、下段が法67条4項の期末配当等タイムラインである。実務では、中央を上から下へ読み、迷ったところだけ左右と下段へ戻ると速い。

最短で押さえる3点
  • 欄19は 欄9 + 欄10 - 欄11 - 欄15 + 欄16 - 欄17 - 欄18 であり、税額控除後の税負担や帳簿価額調整まで織り込む。
  • 欄20は付表一から来るが、3基準のうち最も多い金額を採る。ここを誤ると欄21以降がすべて崩れる。
  • 欄8の税額は B01 欄9外書へ乗せて初めて最終法人税額に反映される。

2. 条文の核

法67条は長いが、留保金課税の骨格は次の5点に集約できる。本文を丸写しするより、どの項がどの欄を支配しているかで読む方が使いやすい。

法67条1項

特定同族会社の各事業年度の留保金額が留保控除額を超えるときは、その超過部分に 10%・15%・20% の特別税率を掛けて通常の法人税額へ加算する。

法67条3項

留保金額は、「所得等の金額のうち留保した金額」から法人税・地方法人税・住民税等を控除して求める。したがって、会計利益と一対一では対応しない。

法67条4項

期末配当等は、基準日等が当期に属し、決議日が事業年度終了の日の翌日から決算確定日までにあるものを対象に、留保した金額から控除・翌期へ加算する。

法67条5項

留保控除額は、所得基準40%、定額基準2,000万円、積立金基準のうち最も多い金額である。付表一はこの比較装置として機能する。

法67条6項

事業年度が1年未満なら、3,000万円・1億円・2,000万円はいずれも月数按分する。短期事業年度では閾値が自動的に下がる。

通達16-1-6 / 16-1-8

期末利益積立金額の見方と、1,000円未満端数の処理が補足される。とくに欄21と1億円相当額の端数関係は手引でも注意点になっている。

所得0でも留保金課税が残りうる理由

国税庁「別表三(一) 記載要領」は、非適格合併の譲渡損益、受取配当益金不算入、外国子会社配当益金不算入、受贈益益金不算入、還付金等益金不算入、繰越欠損金損金算入に加え、措置法上の各種特別控除や超過利子額の損金算入等がある場合、会計上の所得が0でも留保金課税が動き得ると明示している。ここが、通常の「所得課税」と留保金課税が完全には重ならない理由である。

3. 設例で追う

次の設例は、図の読み方をそのまま数値に落としたものである。実務上はもっと多くの欄が動くが、主骨格はこの形で掴める。

項目 金額 メモ
欄9 留保所得金額 120,000,000 B04 欄52の②より
欄10 前期末配当等の額 8,000,000 前期欄11を受ける
欄11 当期末配当等の額 18,000,000 法67条4項の期末配当等のみ
欄15 法人税額等の合計額 29,000,000 欄12 + 欄13 - 欄14
欄16〜18 0 通算調整・帳簿価額減算がない前提
欄19 当期留保金額 81,000,000 120,000,000 + 8,000,000 - 18,000,000 - 29,000,000
欄20 留保控除額 48,000,000 40%基準 48,000,000 / 定額基準 20,000,000 / 積立金基準 12,000,000 の最大値
欄21 課税留保金額 33,000,000 81,000,000 - 48,000,000
欄8 留保金額に対する税額 3,450,000 30,000,000×10% + 3,000,000×15%
設例の読み筋

この設例では、課税留保金額 33,000,000 のうち最初の 30,000,000 が 10% 帯、残り 3,000,000 が 15% 帯に入る。もし事業年度が 9 か月なら、30,000,000 と 100,000,000 はそれぞれ 22,500,000 と 75,000,000 に縮むので、同じ課税留保金額でも税額が変わる。

4. 別表転記マップ

B03(一) は単独で完結せず、複数の別表から値を受け、最後に B01 へ返す。最低限の転記関係だけを並べると次のとおりである。

B03(一)欄 内容 主な参照元
欄9 留保所得金額 B04 欄52の②
欄10 前期末配当等の額 前期 B03(一) 欄11
欄11 当期末配当等の額 法67条4項に従う期末配当等
欄12〜15 税負担控除 B01, B06系, 別表十七(三の六)
欄16〜17 通算法人調整 付表二 欄5 / 欄10
欄18 帳簿価額から減算される金額 付表一 欄19
欄20 留保控除額 付表一 欄34
欄21 課税留保金額 欄19 - 欄20
欄8 留保金額に対する税額 B01 欄9外書へ転記
提出不要と税額0は別概念

欄21が0なら税額欄は0で終わるが、だからといって入口判定や留保控除額比較が不要になるわけではない。判定経過と計算過程を一度辿っておくこと自体が、特定同族会社に関する実務ミスの防止になる。
