外国税額控除・みなし外国税額控除
課税庁側の最強主張 vs 納税者側の最強反論

日中使用料20%みなし外国税額控除を念頭に、法人税法69条25項、実施特例法省令10条、77_22、77_21、サンリオ高裁を踏まえ、争点の本質を「1枚」で把握するための対比メモ。中核テーマは、みなし部分が申告時に外形化されていたか、そして69条25項の `申告書等` 文言をどこまで広く読めるかにある。

作成日: 2026-03-07 / 主DB: jplawdb4 / 関連完全版: FTC中国ロイヤルティ統合完全版

結論

課税庁側の最強主張は、「みなし外国税額控除は、通常FTCよりも強い申告時外形化要件を伴う特別利益であり、省令10条書類が当初申告束から読めないなら、みなし部分は認められない」 というものです。これに対する納税者側の最強反論は、「本件は新規の利益取得ではなく、当初申告で既に外形化されていたみなし部分の補充・訂正であり、77_21型の是正が許される」 という形になります。

1. 争点の本質

この論点の芯は、条約上20%みなしの制度が存在するかそれ自体ではありません。核心は、その制度利益を受けるために、当初申告でどこまで外形化していなければならないか にあります。通常FTCでは改正を通じて証拠書類の一部が保存側へ移りましたが、みなし部分については省令10条がなお添付側に残しています。したがって、争点は「証明できるか」だけでなく、「申告時に法定の形で提示していたか」に移ります。

2. 課税庁側の最強主張 vs 納税者側の最強反論

論点 課税庁側の最強主張 納税者側の最強反論
法的構造 法人税法69条25項はFTC適用の要件を定め、省令10条はみなし外国税額がある場合に `計算明細` と `証明書類` を添付書類群に含める。よって、みなし部分は単なる保存資料ではなく法定添付要件である。 現行69条25項は `確定申告書、修正申告書又は更正請求書` を入口にしており、旧法の純粋な当初申告主義ではない。理論上、後補余地は残っている。
改正史 通常FTCは `添付→保存` に緩和されたのに、みなしだけ省令10条で添付が残った。これはみなしを通常FTCより厳格に扱う立法意思である。 改正後制度は、形式不備があっても一切救済しない構造ではなくなった。`申告書等` 文言は一定の是正可能性をなお残している。
77_22 `77_22` は、みなし納付外国法人税について、申告時の `記載及び添付の要件` を充たさなければ適用不可とした。本件に最も近い。 `77_22` は旧法事案であり、現行69条25項の `申告書等` 構造とは完全には一致しない。旧法の結論をそのまま現行法へ機械的に移植すべきではない。
サンリオ高裁 サンリオ高裁は、海外情報型制度の添付要件を `適用意思の明確化`、`資料の早期・確実な収集`、`適正迅速な判断` のための実体的ゲートとみた。しかも平成23年改正後でも、未添付分の制度選択・適用範囲拡張を広く認めていない。 サンリオ高裁はCFC適用除外事案であり、FTCの直接先例ではない。FTCは二重課税排除制度であり、CFCの適用除外と同列に硬直化すべきではない。
69条28項 69条28項は保存書類欠缺の限定救済であり、添付漏れ一般の救済ではない。したがって省令10条の未添付はここで救えない。 28項自体は直接使いにくいが、立法が証拠面の欠缺に一定の柔軟性を認めていることの傍証にはなる。少なくとも形式のみで全面否認すべきでない。
情報の非対称性 みなし部分は実納付税額と異なり、課税庁から外から把握しにくい。だからこそ省令10条で申告時開示を要求している。後から出せるなら制度趣旨が失われる。 既に別表六で国・所得区分・税額・みなし欄の記載があり、申告時に一定の外形は示されていたなら、後提出資料は新規取得ではなく既存請求の補強にすぎない。
本件への結論 別表六だけで、省令10条の `計算明細` と `証明書類` の内容が申告束から読めないなら、みなし部分は否認。少なくとも追加10%部分は落とす。 当初申告の別表六(四)等から、みなし部分の存在・対象国・対象所得・計算骨格が読めるなら、`77_21` 型に寄せて、更正の請求等での是正余地を主張できる。

3. 課税庁側の最強ストーリー

  1. 通常FTCは改正で緩和されたが、みなしFTCだけは省令10条で添付を残した。
  2. これは、みなし部分を申告時に外形化させる制度だという意味である。
  3. `77_22` はその趣旨を正面から採っている。
  4. サンリオ高裁も、残存添付要件を実体的ゲートとして扱い、平成23年改正後でも未添付分の後出し拡張を認めていない。
  5. よって、本件で省令10条書類が当初申告束に現れていないなら、みなし部分は不可である。

4. 納税者側の最強ストーリー

  1. 現行69条25項は `申告書等` を入口にしており、旧法の完全な当初申告主義ではない。
  2. `77_21` は、申告時に外形があれば訂正を認める。
  3. 省令10条は専用様式名を要求しているのではなく、内容を要求している。
  4. したがって、別表六や添付済資料から、みなし部分の存在・対象・計算骨格が既に示されていたなら、後提出資料は新規請求ではなく補強資料にすぎない。
  5. よって、全面否認ではなく、少なくとも `77_21` 型の訂正余地を認めるべきである。

5. 私の評価

課税庁優勢の場面

  • 当初申告で、みなし部分であること自体が読めない
  • 条約根拠も計算骨格も見えない
  • 別表六はあるが、通常FTCなのかみなしFTCなのか不明
  • 後提出資料により、はじめて追加10%部分が立ち上がる

納税者が戦える場面

  • 当初申告で、別表六(四)等にみなし欄・対象国・対象所得・税額骨格が出ている
  • 後提出資料は、その補充説明にとどまる
  • 新しい利益の追加ではなく、既存請求の是正と言える
  • `77_21` 型の「外形化済みの誤算訂正」に寄せられる

最終的な一文

課税庁側の最強主張は「みなしFTCは申告時外形化を要する特別利益だ」であり、納税者側の最強反論は「本件は新規取得ではなく、当初申告で外形化済みの請求の補充・訂正だ」です。
