1. 全体の流れ（3分で理解）

今の状態と、これからやることの全体像です。

やることは2ステップだけ。Power Queryで「まとめて」、ピボットで「見る」。

2. 始める前に確認すること

Excelのバージョン

バージョンPower Query
Excel 365 / 2021 / 2019標準搭載（そのまま使える）
Excel 2016標準搭載（そのまま使える）
Excel 2013無料アドインをインストール
Excel 2010以前使えない

シートの構造を揃えておく

各シートの1行目が見出し（ヘッダー）になっていること。

A: 会社名B: 売上C: 利益D: ...
A社1,000100...
B社2,000200...
5シートとも列の順番と見出しの名前が同じなら完璧。多少違っても Power Query で調整できます。
見出しの上に「2024年度 売上一覧」みたいなタイトル行がある場合は、先に削除するか、Power Query内で除外します（後述）。

3. Power Query で5シートを結合する

手順（クリックだけでOK）

1

データ タブをクリック

Excelの上部メニューバーにあります

2

データの取得 → ファイルから → Excelブックから

Excel 2016では「新しいクエリ」という名前の場合もあります

3

今開いているファイル自身を選択して「インポート」

4

「ナビゲーター」画面が出る → 5つのシートすべてにチェックを入れる

「複数のアイテムの選択」にチェックを入れると複数選択できます

5

右下の データの変換 をクリック

「読み込み」ではなく「データの変換」を押すのがポイント！

ここで「読み込み」を押すと5つバラバラのまま読み込まれます。必ず「データの変換」を押してください。
6

Power Query エディタが開く（別ウィンドウ）

左側に5つのクエリが一覧表示されます

7

上部メニューの ホーム → クエリの追加 → クエリを新規クエリとして追加

8

「3つ以上のテーブル」タブを選択

左側の「利用可能なテーブル」から5つ全部を 「追加 >>」 ボタンで右側に移す → OK

9

結合されたプレビューが表示される → 内容を確認

500行（100社×5期）になっているはず

10

左上の 閉じて読み込む をクリック

Excelに新しいシートが追加され、結合データが表示されます

これで Power Query の作業は完了です！

どの期のデータかわからなくない？

Power Query が自動で「ソース」列（どのシートから来たか）を追加してくれます。

ソース会社名売上利益
1期A社1,000100
1期B社2,000200
2期A社1,200150
............
もし「ソース」列がない場合は、Step 7の前に各クエリに「カスタム列の追加」で期を手動追加できます。ただし通常は自動で付きます。

もしタイトル行や空行がある場合

Power Query エディタ内で簡単に対処できます：

  • 不要な行を削除：ホーム → 行の削除 → 上位の行の削除
  • 1行目を見出しに：ホーム → 1行目をヘッダーとして使用
  • 空行を除外：列のフィルターボタン → 「null」のチェックを外す

4. ピボットテーブルで分析する

ピボットテーブルとは？

縦長のデータをドラッグ&ドロップだけで好きな表に組み替えられる機能です。

数式は一切書きません。

手順

1

Power Query で作った結合テーブルのどこかのセルをクリック

2

挿入 タブ → ピボットテーブル をクリック

3

「新しいワークシート」を選んで OK

4

右側に「ピボットテーブルのフィールド」パネルが表示される

ここからがドラッグ&ドロップです

5

フィールドを配置する（ドラッグで下の4つのエリアに入れる）：

配置した瞬間にテーブルが自動生成されます。数式は不要です。

よく使う分析パターン

パターン①：会社別×期別の売上一覧

配置場所入れるフィールド
行会社名
列ソース（期）
値売上

パターン②：期別の合計・平均

配置場所入れるフィールド
行ソース（期）
値売上（合計と平均の両方入れる）
「値」に同じフィールドを2回入れて、片方を「平均」に変更できます。値フィールドをクリック →「値の集計方法」→「平均」

パターン③：売上ランキング

配置場所入れるフィールド
行会社名
値売上（合計）
フィルターソース（期）→ 特定の期だけ選択

→ 売上列をクリックして降順ソートすればランキング完成

グラフも一瞬で作れる

1

ピボットテーブルのどこかをクリック

2

ピボットテーブル分析 → ピボットグラフ

3

好きなグラフの種類を選んで OK

ピボットテーブルのフィールド配置を変えると、グラフも自動で変わります。

5. データを更新したいとき

元のシートのデータを修正したら：

1

データ タブ → すべて更新

これだけで Power Query の結合もピボットテーブルも全部更新されます。

VLOOKUPの数式を直す必要も、コピペをやり直す必要もありません。

6. よくあるトラブル

Q. 「データの取得」メニューが見つからない

→ Excel 2013以前の可能性。MicrosoftからPower Queryアドインを入れるか、Excel 365にアップグレード。

Q. 結合したら列がズレる

→ シート間で見出しの名前が違う可能性。Power Query エディタで列名を統一してから結合。

Q. 数字が「合計」じゃなく「個数」になる

→ ピボットテーブルの値フィールドをクリック →「値フィールドの設定」→「合計」に変更。
元データにテキストや空白が混じっていると「個数」になりがちです。

Q. Power Query エディタを再度開きたい

→ データ → クエリと接続（右側にパネルが出る）→ クエリをダブルクリック。
