第1層　日中租税条約（条約）

条約 所得に対する租税に関する二重課税の回避及び脱税の防止のための日本国政府と中華人民共和国政府との間の協定（1983年署名・1984年発効）

条約 第10条（配当）
第10条1項
一方の締約国の居住者である法人が他方の締約国の居住者に支払う配当に対しては、当該他方の締約国において租税を課することができる。
第10条2項 ★ 源泉税率上限
1の配当に対しては、これを支払う法人が居住者とされる締約国においても、当該締約国の法令に従つて租税を課することができる。その租税の額は、当該配当の受領者が当該配当の受益者である場合には、当該配当の額の十パーセントを超えないものとする。
この2の規定は、配当に充てられる利得についての当該法人に対する課税に影響を及ぼすものではない。
→ 23条3項(a)への接続: 10条2項の「配当」に対して、中国側が課税した（又は課税しえた）税額について、日本側でみなし税率を適用する際の前提規定。「第十条2の規定が適用される配当については」という文言で23条3項(a)から直接参照される。
第10条3項（配当の定義）
この条において、「配当」とは、株式その他利得の分配を受ける権利（信用に係る債権を除く。）から生ずる所得及びその他の持分から生ずる所得であつて分配を行う法人が居住者とされる締約国の税法上株式から生ずる所得と同様に取り扱われるものをいう。
第10条4項（PE帰属所得の除外）
1及び2の規定は、一方の締約国の居住者である配当の受益者が、当該配当を支払う法人が居住者とされる他方の締約国において当該他方の締約国内にある恒久的施設を通じて事業を行い〔…〕当該配当の支払の基因となつた株式その他の持分が当該恒久的施設〔…〕と実質的な関連を有するものであるときは、適用しない。この場合には、第七条又は第十四条の規定を適用する。
第10条5項（域外配当への課税禁止）
一方の締約国の居住者である法人が他方の締約国から利得又は所得を取得する場合には、当該他方の締約国は、当該法人の支払う配当及び当該法人の留保所得については〔…〕いかなる租税も課することができず、また、当該留保所得に対して租税を課することができない。
▼10条2項の税率を前提として
条約 第23条（二重課税の除去）
第23条2項 ★ 日本側FTCの基本規定
日本国以外の国において納付される租税を日本国の租税から控除することに関する日本国の法令に従い、
(a) 日本国の居住者がこの協定の規定に従つて中華人民共和国において租税を課される所得を中華人民共和国において取得する場合には、当該所得について納付される中国の租税の額は、当該居住者に対して課される日本国の租税の額から控除する。ただし、控除の額は、日本国の租税の額のうち当該所得に対応する部分を超えないものとする。
(b) 中華人民共和国において取得される所得が、中華人民共和国の居住者である法人によりその議決権のある株式又はその発行済株式の少なくとも二十五パーセントを所有する日本国の居住者である法人に対して支払われる配当である場合には、日本国の租税からの控除を行うに当たり、当該配当を支払う法人によりその所得について納付される中国の租税を考慮に入れるものとする。
→ 国内法への接続ポイント: 「日本国の法令に従い」が、法人税法69条への橋渡し。FTCの計算方法・手続は国内法に委ねるという条約上の構造。ただし、何を控除するか（控除の存在と範囲）は条約が定める。
第23条3項 ★★ みなし外国税額控除（Tax Sparing）本体
2(a)に規定する控除の適用上、中国の租税は、次の率で支払われたものとみなす。

(a) 第十条2の規定が適用される配当については、中華人民共和国の合弁企業が支払う配当である場合には十パーセント、その他の配当である場合には二十パーセント

(b) 第十一条2の規定が適用される利子については十パーセント

(c) 第十二条2の規定が適用される使用料については二十パーセント
→ 3項の機能: 中国が実際に課税した税率にかかわらず、配当10%/20%、利子10%、使用料20%の税率で「支払われたものとみなす」。中国が投資奨励税制で税率0%としていても、日本側では上記のみなし税率で外国税額控除を計算できる。これがtax sparingの核心。
→ 10条2項との接続: 3項(a)は「第十条2の規定が適用される配当については」と明示的に10条2項を参照し、配当の源泉税上限（10%）の枠内で、みなし税率（合弁10%/その他20%）を設定する。
第23条4項 ★ 奨励措置のみなし（マッチング・クレジット）
2に規定する控除の適用上、「納付される中国の租税」には、次のいずれかのものに従つて免除、軽減又は還付が行われないとしたならば納付されたとみられる中国の租税の額を含むものとみなす。

(a) 中華人民共和国合弁企業所得税法第五条及び第六条の規定並びに中華人民共和国合弁企業所得税法施行細則第三条の規定
(b) 中華人民共和国外国企業所得税法第四条及び第五条の規定
(c) この協定の署名の日の後に中華人民共和国の法令に導入される中華人民共和国の経済開発を促進するための他の同様な特別の奨励措置で両締約国の政府が合意するもの
→ 3項と4項の関係: 3項は「みなし税率」（税率自体をみなす）、4項は「みなし納付」（奨励措置で免除された税額を納付されたものとみなす）。両方がFTCの「外国法人税の額」に加算される。
▼ ▼ ▼条約23条2項「日本国の法令に従い」→ 国内法へ

第2層　租税条約等実施特例法（法律）

法律 租税条約等の実施に伴う所得税法、法人税法及び地方税法の特例等に関する法律（昭和44年法律第46号）

法律 条約と国内法の「橋渡し」法律

実施特例法は、租税条約の国内実施のための「橋渡し」法律である。ただし、みなし外国税額控除について法人税法69条を読み替える固有の規定は置いていない。みなしFTCは、条約の規定に基づき、法人税法69条1項の「外国法人税の額」にみなし税額が含まれるという構成をとる。

第7条（租税条約に基づく合意があつた場合の更正の特例）第2項 — 抜粋
相手国等の法令に基づき、居住者又は内国法人に係る当該相手国等の租税〔…〕の課税標準等につき更正又は決定に相当する処分があつた場合において〔…〕財務大臣と当該相手国等の権限ある当局との間の当該租税条約に基づく合意が行われたことにより、〔…〕内国法人の各事業年度の国外所得金額のうちに増額されるものがあり〔…〕法人税法第六十九条〔…〕
第12条 ★ 省令への包括委任
第二条から前条までに定めるもののほか、租税条約等の実施及びこの法律の適用に関し必要な事項は、総務省令、財務省令で定める。
→ 省令への委任構造: 第12条が、実施特例法省令（第3層）に対する包括的な委任規定。みなし外国税額控除の添付要件は、この委任に基づいて省令第10条で規定される。
→ 憲法84条との緊張: 判決12672は「実特法12条が課税要件等の定めを省令に委ねたものと解することはできない」と判示。この包括委任が「手続規定」にとどまるのか「適用要件」にまで及ぶのかが、添付要件問題の核心。

第3層　実施特例法省令（財務省令）

省令 租税条約等の実施に伴う所得税法、法人税法及び地方税法の特例等に関する法律の施行に関する省令

省令 みなし外国税額の定義と添付要件
第1条第9号 ★ みなし外国税額の定義
九　みなし外国税額　相手国等の法律の規定又は当該相手国等との間の租税条約の規定により軽減され又は免除された当該相手国等の租税の額で、当該租税条約の規定に基づき納付したものとみなされるものをいう。
→ 条約23条3項・4項との接続: 条約が「支払われたものとみなす」（3項）・「納付されたとみられる中国の租税の額を含むものとみなす」（4項）と定めた税額が、ここで「みなし外国税額」として国内法上の概念に定義される。
第10条 ★★ みなし外国税額の控除の申告手続等
居住者又は内国法人が所得税法第九十五条、法人税法第六十九条又は地方税法〔…〕の規定による外国税額の控除を受けようとする場合において、〔…〕法人税法第六十九条第一項に規定する外国法人税の額〔…〕のうちにみなし外国税額があるときは、次に掲げる書類には、控除を受けるべきみなし外国税額の計算の明細を記載した書類及び当該みなし外国税額を証明する書類を含むものとする。

二　法人税法第六十九条第二十五項から第二十七項まで又は第三十一項の規定により同条第二十五項に規定する申告書等〔…〕に添付すべき書類
→ 法人税法69条25項への接続: 省令10条は、法人税法69条25項が要求する「添付すべき書類」に、みなし外国税額に固有の2種類の書類（計算明細書＋証明書類）を「含むものとする」と規定する。つまり、69条25項の添付要件の内容を、みなしFTCに特化して拡張する構造。
→ 構造上の帰結: みなし外国税額の証明書類が添付されなければ、69条25項の「書類の添付があり…場合に限り、適用する」が不充足となり、控除が否定される。
▼省令10条が「含むものとする」書類の添付先 → 法人税法69条25項

第4層　法人税法（法律）

法律 法人税法（昭和40年法律第34号）第69条（外国税額の控除）

法律 FTCの基本規定・適用要件・救済規定
第69条1項 ★ FTCの基本規定
内国法人が各事業年度において外国法人税を納付することとなる場合には、当該事業年度の所得の金額につき第66条第1項から第3項までの規定を適用して計算した金額のうち当該事業年度の国外所得金額に対応するものとして政令で定めるところにより計算した金額（控除限度額）を限度として、その外国法人税の額（〔…〕控除対象外国法人税の額）を当該事業年度の所得に対する法人税の額から控除する。
→ みなしFTCとの接続: 条約23条3項・4項により「納付したものとみなされる」税額は、この「外国法人税を納付することとなる場合」の「外国法人税の額」に含まれる。省令第1条9号で「みなし外国税額」として定義されたものが、ここに流入する。
→ 施行令への委任: 「政令で定めるところにより計算した金額（控除限度額）」→ 施行令142条。
第69条25項 ★★★ 適用要件 —「場合に限り、適用する」
第1項の規定は、確定申告書、修正申告書又は更正請求書に第1項の規定による控除を受けるべき金額及びその計算に関する明細を記載した書類並びに控除対象外国法人税の額の計算に関する明細その他の財務省令で定める事項を記載した書類の添付があり、かつ、控除対象外国法人税の額を課されたことを証する書類その他の財務省令で定める書類を保存している場合に限り、適用する。

この場合において、第1項の規定による控除をされるべき金額の計算の基礎となる控除対象外国法人税の額その他の財務省令で定める金額は、税務署長において特別の事情があると認める場合を除くほか、当該明細書に記載された金額を限度とする。
→ 添付と保存の二重要件:
①「添付があり」 — 申告書に書類を物理的に添付すること
②「保存している」 — 証拠書類を手元に保管していること
③「場合に限り、適用する」 — ①②の両方が充足されない限り、69条1項のFTCは適用されない

→ 施行規則への委任: 「財務省令で定める事項を記載した書類」→ 施行規則29条の4第1項。「財務省令で定める書類」→ 施行規則29条の4第2項。
→ 省令10条との接続: みなし外国税額がある場合、ここで「添付すべき書類」に省令10条の2書類が「含まれる」ことになる。
第69条28項 ★ 救済規定 — 「保存」のみ
税務署長は、第1項から第3項又は第18項の規定による控除をされるべきこととなる金額の全部又は一部につき前3項に規定する財務省令で定める書類の保存がない場合においても、その書類の保存がなかったことについてやむを得ない事情があると認めるときは、その書類の保存がなかった金額につき第1項から第3項又は第18項の規定を適用することができる。
→ 構造的欠落: 28項は「保存がない場合」の救済のみを定める。「添付がない場合」の救済規定は存在しない。
→ 帰結: 書類を保存していたが添付を忘れた場合、25項の「場合に限り」により控除は否定され、28項による救済も得られない。みなしFTCの証明書類を添付し忘れた場合も同様。
▼69条1項「政令で定めるところにより計算した金額」→ 施行令

第5層　法人税法施行令（政令）

政令 法人税法施行令（昭和40年政令第97号）第二款 外国法人税額の控除

政令 FTCの計算構造とみなし納付額の取扱い
第141条（外国法人税の範囲）— 抜粋
外国の法令に基づき外国又はその地方公共団体により法人の所得を課税標準として課される税を外国法人税とする。
→ みなし外国税額との関係: みなし外国税額は「実際に課された」税ではないが、条約の規定により「納付したものとみなされる」ため、141条の「外国法人税」に擬制的に含まれる。この擬制は条約に基づくもので、施行令自体は擬制規定を置いていない。
第142条1項（控除限度額の計算）— 要旨
法第69条第1項に規定する控除限度額は、内国法人の各事業年度の所得に対する法人税の額に、当該事業年度の所得金額のうちに当該事業年度の調整国外所得金額の占める割合を乗じて計算した金額とする。
第142条の2第3項 ★ みなし納付額の優先適用
租税条約により軽減又は免除された租税につき、みなし納付額が含まれる場合には、当該外国法人税の額は、まずそのみなし納付額から成るものとする。
→ 高率負担判定の特則: 施行令142条の2第1項は、外国法人税のうち「課税標準×35%」を超える部分を控除対象外とする（高率負担排除）。第3項は、みなし納付額がある場合、外国法人税の額は「まずそのみなし納付額から成る」とし、みなし納付額から先に高率負担判定にかけるルールを定める。
→ 実際上の効果: みなし税率（10%〜20%）は通常35%を下回るため、高率負担で排除されることは稀。ただし、みなし税率20%＋実際課税のある場合に合算が35%を超えうるケースでは機能する。
第142条の2第8項5号（条約限度税率超過部分の排除）— 要旨
租税条約に定める限度税率を超える部分の外国法人税の額は、控除対象外国法人税の額に含まれない。

第29条の4第1項 ★ 添付すべき計算明細書類
法第69条第25項に規定する財務省令で定める事項を記載した書類は、次に掲げる事項を記載した書類とする。

一　外国法人税が法第69条第1項の対象であることの説明及び控除対象外国法人税額の計算明細
二　外国法人税が減額された場合の減額金額・減額日・減額控除対象外国法人税額の計算明細
三　適格合併等に係る被合併法人等の減額控除対象外国法人税額の明細
四　租税特別措置法第66条の7適用時の個別計算外国法人税額及び控除対象相当額の計算明細
〔…五〜七 省略…〕
八　第四号又は第六号の税に係る申告書写し等及び納付証明書
第29条の4第2項 ★ 保存すべき証拠書類
法第69条第25項に規定する財務省令で定める書類は、次に掲げる書類とする。

一　外国税を課されたことを証する申告書写し又は同等の書類及び納付証明書
二　地方税法施行令の規定適用時の限度額計算基礎を証する地方税申告書等
→ 省令10条との接続: 実施特例法省令10条は、上記の29条の4第1項の書類に「みなし外国税額の計算明細書」と「みなし外国税額を証明する書類」を「含むものとする」と規定する。つまり:
29条の4第1項の書類 ＋ 省令10条の追加2書類 ＝ みなしFTCを受けるために添付すべき書類の全体

→ 69条25項への還流: これらの書類全てが「添付」されなければ、69条25項の「場合に限り」要件が不充足となる。

添付要件不充足時の「行き止まり」構造

みなし外国税額控除の添付書類（省令10条の2書類＋施行規則29条の4の書類）を期限内に添付しなかった場合、制度上のすべての救済経路が閉鎖される。

【添付忘れの場合の経路探索】

  ① 法人税法69条25項 ←── 「添付があり…場合に限り、適用する」→ 不適用
     │
     ├── ② 69条28項で救済？ ←── 「保存がない場合」のみ対象。「添付」不備は射程外 → 不可
     │
     ├── ③ 更正の請求？ ←── 69条25項後段「記載された金額を限度」→ 記載ゼロなら限度ゼロ → 不可
     │
     ├── ④ やむを得ない事情？ ←── 28項は「保存」の救済のみ。添付の「やむを得ない事情」は規定なし → 不可
     │
     ├── ⑤ 嘆願？ ←── 法的根拠なし。税務署長の事実上の裁量のみ → 不確実
     │
     └── ⑥ 条約の直接適用？ ←── 判決12672は肯定したが12689（控訴審）で覆り、最高裁判断なし → 未確立

→ 制度上の全救済経路が閉鎖 = 条約上の権利の永久喪失

条約→国内法の「権限移譲」と「権利遮断」の非対称性

側面条約が国内法に委ねた範囲条文根拠
何を控除するか 条約が決定。みなし税率（10%/20%）は条約23条3項が定める 条約23条3項
どう計算するか 国内法に委任。「日本国の法令に従い」 条約23条2項柱書 → 施行令142条
どう申告するか 国内法に委任。添付・保存の手続は国内法が規定 69条25項 + 省令10条
申告手続の不備で権利消滅するか 条約は沈黙。国内法は「場合に限り」で消滅させる構造 69条25項 — 条約の明示的授権なし
構造的問題: 条約は「どう計算し申告するか」を国内法に委ねたが、「手続不備で権利自体を消滅させてよい」とまで授権したかは条約の文言からは読み取れない。69条25項の「場合に限り」は、条約が沈黙している領域に国内法が踏み込んで条約上の権利を遮断する効果を持つ。ここに憲法84条と98条2項の構造的緊張が生じる。
