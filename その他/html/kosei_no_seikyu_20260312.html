1. 制度の趣旨と沿革

制度趣旨

更正の請求とは、納税申告書を提出した者が、申告に係る税額等が過大であることに気づいた場合に、税務署長に対し減額更正を求める納税者の権利救済手続である。申告納税制度の下では、納税者が自己の判断と責任において課税標準等及び税額等を計算し申告するが、その過程で生じた不利な過誤を是正する機会を制度的に保障するものである。

沿革

時期内容
昭和21年申告納税方式の導入と同時に更正の請求制度を創設。請求期限は法定申告期限から1月
昭和37年国税通則法の制定。更正の請求を通則法23条に統一的に規定。期限は2月に延長
昭和45年期限を1年に延長。同時に後発的事由による更正の請求（23条2項）を新設
平成23年期限を5年に大幅延長（平成23年12月2日施行）。嘆願制度の解消を企図
排他的救済手段としての性格：更正の請求は、過大納付税額の是正について排他的な救済手段とされる。名古屋高裁昭和52年6月28日判決は、所得税法に更正の請求という救済手段が用意されているにもかかわらず、これによらず民法上の不当利得返還請求を認めると、「右規定は空文化するのみならず、納税者の恣意を許し、課税の公平を期しがたい」とし、現行税法体系を崩すおそれがあると判示した。

2. 基本条文の構造

2-1. 国税通則法23条1項（通常の更正の請求）

【条文趣旨】納税申告書を提出した者は、以下のいずれかに該当する場合に、法定申告期限から5年以内に限り、税務署長に対し更正をすべき旨の請求をすることができる。

請求の要件（1項各号）

号要件内容
1号 税額等が過大 申告書に記載した課税標準等もしくは税額等の計算が国税に関する法律の規定に従っていなかったこと又は当該計算に誤りがあったことにより、納付すべき税額が過大であるとき
2号 純損失等の金額が過少 1号に規定する理由により、申告書に記載した純損失等の金額が過少であるとき、又は純損失等の金額の記載がなかったとき
3号 還付金額が過少 1号に規定する理由により、申告書に記載した還付金の額に相当する税額が過少であるとき、又は還付金の額に相当する税額の記載がなかったとき

請求期限

税目期限
原則（所得税・消費税等）法定申告期限から5年
法人税（純損失等の金額）法定申告期限から10年（H30.4.1以後開始事業年度。それ以前は9年）
贈与税法定申告期限から6年
移転価格税制に係る法人税法定申告期限から6年

手続要件（23条3項）

更正の請求をしようとする者は、更正請求書を税務署長に提出しなければならない。更正請求書には以下を記載する：

  • 更正後の課税標準等又は税額等
  • 更正の請求をする理由
  • 当該請求をするに至った事情の詳細
  • その他参考となるべき事項

また、更正の請求の理由の基礎となる「事実を証明する書類」の添付が必要（平成23年改正で明確化）。

2-2. 国税通則法23条2項（後発的事由による更正の請求）

【条文趣旨】申告時には予想し得なかった後発的事由により課税要件事実に変動が生じた場合に、法定申告期限からの期間制限を超えて更正の請求を認める特例。昭和45年改正で新設。

帰責事由のない納税者に酷な結果となることを防ぐため、以下の3つの後発的事由に限定して、事由発生日から2月以内に更正の請求を認める。

号後発的事由期限
1号 判決等による事実の確定
課税標準等又は税額等の計算の基礎となった事実に関する訴えについての判決（判決と同一の効力を有する和解等を含む）により、その事実が当該計算の基礎としたところと異なることが確定したとき		 判決確定日の翌日から2月
2号 他者の更正・決定に伴う帰属所得の変更
申告等に係る課税標準等又は税額等の計算にあたってその申告等をした者に帰属するものとされていた所得その他課税物件が、他の者に係る国税の更正又は決定により、他の者に帰属するものとされたとき		 更正・決定日の翌日から2月
3号 政令で定めるやむを得ない理由
その他当該申告等の後に生じた通則法施行令6条で定めるやむを得ない理由があるとき		 理由が生じた日の翌日から2月

2-3. 国税通則法施行令6条（後発的事由の具体的内容）

通則法23条2項3号に規定する「政令で定めるやむを得ない理由」を以下の5号で列挙。
号事由具体例
1号 官公署の許可等の処分の取消し
申告等の基礎となった事実に係る官公署の許可その他の処分が取り消されたこと		 農地売買の許可が取り消された場合
2号 契約の解除・取消し
申告等の基礎となった事実に係る契約が、解除権の行使によって解除され、若しくは当該契約の成立後生じたやむを得ない事情によって解除され、又は取り消されたこと		 不動産譲渡後に買主が契約解除権を行使した場合
3号 帳簿書類の押収等の事情消滅
帳簿書類の押収その他やむを得ない事情により、課税標準等又は税額等の計算の基礎となるべき帳簿書類その他の記録に基づいて国税の課税標準等又は税額等を計算することができなかった場合において、その後当該事情が消滅したこと		 査察事件で押収された帳簿が返還された場合
4号 租税条約に基づく相互協議の合意
租税条約の規定に基づき権限のある当局間の協議により、その申告等に係る課税標準等又は税額等につき、その内容と異なる内容の合意が行われたこと		 移転価格税制に関する二国間の相互協議で合意が成立した場合
5号 国税庁長官の法令解釈の変更公表
国税庁長官が法令の解釈について、審査請求若しくは訴えについての裁決若しくは判決に伴って変更し、その旨を公表したことを知ったこと		 通達の解釈変更が公表され、それを知った場合

2-4. 各税法固有の更正の請求の特則

国税通則法は一般法であり、各個別税法にその例外が規定されている場合は、個別税法の規定が優先適用される。以下は主要な特則の一覧である。

税法条文内容期限
法人税法 80条の2 前事業年度の法人税額等の更正・決定又は修正申告に伴い、後続事業年度の税額が過大又は還付税額が過少となった場合 更正等の日から2月以内
所得税法 152条 各種所得の金額の計算の基礎となった事実のうち、無効な行為により生じた経済的成果がその行為の無効であることに基因して失われたこと等 事由発生日から2月以内
所得税法 63条・64条
（152条・153条経由）		 事業廃止後の必要経費（63条）、回収不能債権の必要経費算入（64条）に伴う前年分の更正の請求 事由発生日から2月以内
相続税法 32条 未分割遺産について法定相続分で申告後、遺産分割により取得財産の課税価格が変動した場合（配偶者税額軽減、小規模宅地の特例の適用等を含む） 事由を知った日の翌日から4月以内
消費税法 56条 前課税期間に係る消費税額等の更正等に伴い、当課税期間の税額が過大となった場合 更正等の日から2月以内
通則法23条と個別税法特則の併用：相続税法32条の場合、事由を知った日の翌日から4月以内（相続税法の特則）と、法定申告期限から5年以内（通則法23条1項）のいずれか遅い方が期限となる。両方の規定を併用できる点に留意。

3. 平成23年度税制改正の影響

3-1. 改正の背景と目的

改正前は更正の請求期間が1年と短かったため、期間経過後に税額の誤りに気づいた納税者が法定外の手続として「嘆願書」を提出して課税庁に減額更正を求める実務慣行が広く存在していた。平成23年度改正は、この嘆願制度を解消し、納税者の権利救済と課税の適正化とのバランスを図るとともに、更正の請求期間と増額更正の除斥期間を統一することで制度の簡素化を企図したものである。

3-2. 改正前後の比較

改正前（～H23.12.1）

項目期間
更正の請求1年
増額更正3年（脱税は7年）
減額更正5年
証拠書類添付義務なし
虚偽記載罰則なし

改正後（H23.12.2～）

項目期間
更正の請求5年
増額更正5年（脱税は7年）
減額更正5年
証拠書類事実証明書類の添付義務
虚偽記載罰則1年以下の懲役又は50万円以下の罰金

3-3. 嘆願制度との関係

改正により更正の請求期間と減額更正の除斥期間がいずれも5年に統一されたため、更正の請求期間内に法定の手続で救済を求めることが可能となった。これにより、嘆願書に頼る必要性は理論上大幅に減少した。

ただし、嘆願書は法律上完全に廃止されたわけではなく、5年を経過した後の救済手段としてなお実務上提出されることがある。もっとも、嘆願書は「法律上は納税者の公務員に対する一種の陳述書」にすぎず、税務署長にはこれに基づいて更正を行う法的義務はない。

3-4. 適用時期・経過措置

  • 適用開始：平成23年12月2日以後に法定申告期限が到来する国税について適用
  • 経過措置：同日前に法定申告期限が到来した国税については、従前の1年の期間制限が適用される。ただし、課税庁は「更正の申出書」を受理し、調査の上、必要な場合は減額更正を行うこととされた

4. 更正の請求が認められる要件

4-1. 「税額等が過大」の意味

更正の請求が認められるためには、単に「税額が減少する」というだけでは足りない。以下の2つの事由のいずれかに起因して税額等が過大となっていることが必要である。

(a) 「国税に関する法律の規定に従っていなかったこと」

税法の強行規定に違反した計算がなされていた場合を指す。例えば：

  • 課税対象でない所得を課税所得として申告した場合
  • 非課税規定の適用を見落とした場合
  • 税法の要件を満たしているにもかかわらず、法定の控除を適用しなかった場合（ただし選択的規定の場合は除く）
  • 小規模宅地の特例の対象要件を満たさない土地に誤って適用した場合（要件誤認の是正）

(b) 「計算に誤りがあったこと」

法律の適用は正しいが、算術的・事実的な計算誤りがあった場合を指す。例えば：

  • 収入金額や必要経費の転記ミス
  • 税率の適用誤り
  • 所得控除額の計算間違い
  • 税額控除の計算上の誤り

4-2. 後発的事由の類型（通則法23条2項・施行令6条）

後発的事由の体系図 ┌─ 通則法23条2項 ─────────────────────┐ │ │ │ 1号：判決等による事実の確定 │ │ 2号：他者への帰属所得の変更 │ │ 3号：政令で定めるやむを得ない理由 │ │ │ │ │ └─ 施行令6条 ─────────────┐ │ │ │ 1号：官公署処分の取消し │ │ │ │ 2号：契約の解除・取消し │ │ │ │ 3号：帳簿押収等の事情消滅 │ │ │ │ 4号：租税条約の相互協議合意 │ │ │ │ 5号：法令解釈変更の公表 │ │ │ └────────────────┘ │ │ │ │ ※ すべて事由発生日から2月以内 │ └──────────────────────────┘

4-3. 立証責任

更正の請求においては、立証責任は納税者側にある。請求者は、税額等が過大である旨及びその理由の基礎となる事実を「事実を証明する書類」の添付により証明しなければならない（平成23年改正で明確化）。

5. 更正の請求が認められない場合の類型

5-1. 選択的規定における選択の変更

税法上、複数の処理方法が認められている場合（選択的規定）に、一方を選んで適法に申告した後、より有利な他方の処理方法に変更することを目的とした更正の請求は認められない。

典型例：不動産取得税・登録免許税の処理方法
  • 業務用不動産の取得に伴う不動産取得税・登録免許税は、「取得価額に算入する」方法と「損金（必要経費）として処理する」方法の両方が税法上認められている
  • 取得価額に算入して申告した後に、損金処理の方が有利であったと気づいても、両方とも「法律の規定に従った」計算であるため、更正の請求は認められない
理由：申告書の計算が「法律の規定に従っていなかった」にも「計算に誤りがあった」にも該当しないため。

5-2. 特例措置の意図的不適用

税制上の特例措置（租税特別措置法の各種特例等）の適用を、申告時に意図的に選択しなかった場合、後からその適用が有利であったと判明しても更正の請求は認められない。

  • 小規模宅地の特例を適用せずに申告 → 後から適用すれば有利と判明 → 更正の請求不可
  • 配偶者の税額軽減を適用せずに申告 → 同上

ただし、これらは「当初申告要件」が廃止された規定については、取扱いが異なり得る（後述）。

5-3. 事実の評価の変更

申告時に把握していた事実関係に変化がないにもかかわらず、その法的評価や経済的評価を事後的に変更することを理由とする更正の請求は認められない。

  • 土地の評価方法を路線価方式から倍率方式に変更するなど、評価手法の変更
  • 非上場株式の評価額を申告後に有利な方法で再計算すること

5-4. 法律解釈の変更

申告時に採用した法律解釈が、後に判例等により誤りであったことが明らかになった場合でも、原則として通常の更正の請求（23条1項）の対象とはならない。ただし、施行令6条5号により、国税庁長官の法令解釈が裁決・判決に伴って変更され公表された場合は、後発的事由による更正の請求（23条2項3号）が認められ得る。

5-5. 申告時に適用可能だった規定の不適用（当初申告要件）

一定の租税特別措置法の規定について、確定申告書に適用を受ける旨の記載・明細書添付が要件とされている場合（「当初申告要件」）、当初の確定申告で適用を受けなかった規定を、後から更正の請求により適用することは原則として認められない。

平成23年改正による当初申告要件の見直し：

平成23年12月2日以後の所得税について、一部の規定で当初申告要件が廃止された。例えば：

  • 給与所得者の特定支出の控除の特例
  • 純損失の繰越控除
  • 雑損失の繰越控除

これにより、これらの規定については更正の請求で適用を求めることが可能となった。

5-6. まとめ：認否の判定フロー

更正の請求の認否判定 (1) 申告の計算は法律の規定に従っていたか？ │ ├─ 従っていなかった → 更正の請求 ○（法律違反要件充足） │ └─ 従っていた ─→ (2) 計算に誤りがあったか？ │ ├─ あった → 更正の請求 ○（計算誤り要件充足） │ └─ なかった → 更正の請求 × （適法な申告の事後的変更は不可） 例：選択的規定の変更、 特例の意図的不適用、 評価方法の変更

6. 更正の請求と他の制度との関係

6-1. 修正申告との関係

更正の請求修正申告
方向税額の減少を求める税額の増加を行う
性質納税者の権利行使納税者の義務履行
効果税務署長の処分（更正）を促す申告自体が税額を確定
期限原則5年（税目により異なる）期限なし（いつでも可能）
不服申立て更正の請求の却下・棄却に対し可能修正申告に対しては不可
加算税なし過少申告加算税の対象となり得る
重要な相違点：修正申告は納税者の行為自体が税額を確定させるため、不服申立ての対象とならない。一方、更正の請求は税務署長の処分を促す行為であり、その処分に対して不服申立てが可能。したがって、税務調査で増額を求められた場合、安易に修正申告に応じるよりも更正処分を受けた方が、不服申立ての機会が確保される。

6-2. 職権更正（国税通則法24条）との関係

制度の概要

税務署長は、納税申告書の提出があった場合において、その計算が法律の規定に従っていなかったとき、その他課税標準等又は税額等が調査したところと異なるときは、職権で課税標準等又は税額等を更正する（通則法24条）。これには増額更正と減額更正の両方が含まれる。

更正の請求との関係

  • 更正の請求は、税務署長に減額更正の職権発動を促す手続という側面を有する
  • 更正の請求がなくとも、税務署長は調査により誤りを発見・確認した場合、減額更正をする義務があるとされる
  • ただし、「調査対象とするかどうか」の判断には裁量がある（東京高裁平成3年1月24日判決参照）

期間制限

更正の種類除斥期間
増額更正原則5年（偽り等の場合は7年）
減額更正原則5年（通則法70条2項）

6-3. 減額更正の嘆願

嘆願の法的性質

嘆願（嘆願書の提出）は、法律に規定のない事実上の手続であり、「法律上は納税者の公務員に対する一種の陳述書」にすぎない。税務署長は嘆願書を受理する法的義務も、それに基づいて更正を行う法的義務も、回答を行う義務もない。

嘆願が用いられる場面

  • 更正の請求期間（5年）を経過した後に税額の誤りに気づいた場合
  • 通常の更正の請求の要件を充足しない場合（選択的規定の変更等）

嘆願と職権更正の関係

嘆願書は直接の更正理由とはならないが、課税庁が職権更正（通則法24条）の端緒として調査を開始する契機とはなり得る。嘆願書の内容が減額更正の要件を満たす場合、課税庁は「適切な手続を経て減額更正を検討すべき」と解される。

7. 実務上の留意点

7-1. 更正の請求書の記載事項と添付書類

  • 更正後の課税標準等・税額等を具体的に計算して記載
  • 更正の請求の理由を明確・具体的に記載
  • 理由の基礎となる事実を証明する書類の添付（平成23年改正後は義務化）
  • 虚偽記載には罰則あり（1年以下の懲役又は50万円以下の罰金）

7-2. 税務署長の対応

  • 更正の請求に対し、税務署長は調査した上で処分を行う
  • 請求に理由がある場合 → 減額更正を行う
  • 請求に理由がない場合 → 更正をすべき理由がない旨を記載した通知書を送付
  • 通知書に対しては不服申立てが可能（再調査請求→審査請求→訴訟）

7-3. 各税目の請求期限一覧

税目通常の更正の請求特則による更正の請求

法人税（純損失等）法定申告期限から10年同上
相続税法定申告期限から5年相続税法32条：事由を知った日から4月

消費税法定申告期限から5年消費税法56条：更正等から2月

7-4. 更正の請求を検討する際のチェックポイント

  1. 期限の確認：法定申告期限からの経過年数を確認。5年を超えている場合は後発的事由の有無を検討
  2. 要件の確認：「法律の規定に従っていなかった」又は「計算に誤りがあった」のいずれかに該当するか
  3. 選択的規定の確認：適法な選択の事後変更でないか（適法→適法の変更は不可）
  4. 立証資料の準備：事実を証明する書類が十分に準備できるか
  5. 個別税法特則の確認：通則法23条とは別に、各税法固有の特則が適用できないか
  6. 嘆願の検討：更正の請求が認められない場合でも、職権更正を促す嘆願の余地はあるか
