1. 国税通則法23条1項の更正の請求の要件

1-a. 条文テキストと基本構造

国税通則法23条1項（通常の更正の請求） 納税申告書を提出した者は、次の各号のいずれかに該当する場合には、当該申告書に係る国税の法定申告期限から5年以内に限り、税務署長に対し、その申告に係る課税標準等又は税額等につき更正をすべき旨の請求をすることができる。
一　当該申告書に記載した課税標準等若しくは税額等の計算が国税に関する法律の規定に従っていなかったこと又は当該計算に誤りがあったことにより、当該申告書の提出により納付すべき税額（中略）が過大であるとき。

更正の請求の3要件

  1. 実体要件（原因）：税額等の計算が「国税に関する法律の規定に従っていなかったこと」又は「計算に誤りがあったこと」
  2. 結果要件：納付すべき税額が過大であること
  3. 期間要件：法定申告期限から5年以内（平成23年改正後。改正前は1年以内）

1-b. 「法律の規定に従っていなかったこと又は計算に誤りがあったこと」の解釈

この要件は、二つの類型を含む。

  • 「国税に関する法律の規定に従っていなかったこと」：法令の適用・解釈を誤った場合。例えば、税率の適用誤り、所得区分の誤り、控除規定の適用誤り等。
  • 「当該計算に誤りがあったこと」：法令の適用自体は正しいが、計算過程に算術的・事実認定上の誤りがあった場合。為替レートの換算誤り、転記ミス、集計ミス等。

重要なのは、いずれも「税額等の計算」に関する誤りであり、手続的な選択や意思表示の問題とは区別されるという点である。

1-c. 手続要件の不備（書類添付漏れ等）は「計算の誤り」に該当するか

核心的論点

手続要件の不備（例：添付書類の漏れ、明細記載の欠如）が通則法23条1項の「計算の誤り」に該当するかは、場面によって結論が分かれる。

場面通則法23条の該当性根拠
FTCの適用を選択して申告したが、計算・記載に誤りがあった 該当する（更正の請求可） 裁決事例集No.77-21
FTCの適用を選択しなかった（記載・添付なし） 改正前：不該当（不可）
改正後：該当する（可）		 平成23年改正による当初申告要件廃止
FTCの一部を選択して申告したが、別の外国法人税について添付・記載を完全に欠く 不該当（不可） 裁決事例集No.77-22
FTCの適用意思はあったが失念して記載・添付しなかった 改正前：不該当（やむを得ない事情なければ不可）
改正後：該当する（可）		 裁決事例集No.65-34 + 平成23年改正

2. 外国税額控除における更正の請求 ― 裁決例の分析

2-a. 「選択」と「計算の誤り」の境界 ― 裁決事例集No.77の2件の対比

裁決事例集No.77-21（平21.5.20裁決）― 更正の請求を認容

事案：納税者は、D社・E社からの配当に係るみなし外国税額控除を含む外国税額控除を確定申告書に記載して申告した。しかし、為替レートの換算誤り、別表間の転記誤り、租税条約の解釈誤り（みなし間接外国税額控除の上限に関する誤解）などにより、外国税額控除額が過少に計算されていた。

判断：審判所は以下の解釈を示した。

  • 法69条16項（現25項）の「控除をされるべき金額」とは、内国法人が外国税額控除の適用を選択したと認められる範囲内において、誤りを是正した上で正当に算定される金額と解するのが相当
  • 当初申告において外国税額控除の対象に選択して申告記載した事項については、記載金額又は計算に誤りがあったため控除金額が過少になっている場合、基本的には控除額の増額を認めるべき
  • 本件の各誤りは、外国税額控除の適用を選択した後の控除税額の算出過程における誤りであるから、通則法23条1項の「計算が国税に関する法律の規定に従っていなかったこと又は当該計算に誤りがあったこと」に該当する

裁決事例集No.77-22（平21.5.20裁決）― 更正の請求を不認容

事案：納税者は、外国子会社E社に係る「間接納付した控除対象外国法人税の額」については申告書に記載・添付していたが、E社の「みなし納付外国法人税の額につき間接納付したとみなされる控除対象外国法人税の額」については、別表六（五）への記載も別表六（五の三）の添付も全くなかった。税理士が当該控除の適用を失念していた。

判断：審判所は以下の判断を示した。

  • みなし納付外国法人税に係る外国税額控除については、記載及び添付の要件を充足していない
  • 失念は「やむを得ない事情」に該当しない
  • 記載及び添付の要件を充足しない以上、外国税額控除の適用を受けることができない
  • よって、「納税申告書に記載した税額等の計算が国税に関する法律の規定に従っていなかったこと又は当該計算に誤りがあったこと」に該当しない

2件の対比から導かれる法理

区別基準No.77-21（認容）No.77-22（不認容）
FTC適用の「選択」の有無 選択あり（記載・添付あり） 選択なし（記載・添付なし）
誤りの性質 選択後の計算過程の誤り 手続要件（記載・添付）の不履行
通則法23条該当性 「計算の誤り」に該当 「計算の誤り」に該当しない

結論：FTCの適用を「選択」した上での計算誤りは通則法23条の「計算の誤り」に該当するが、FTCの適用自体を選択しなかった（記載・添付の手続要件を充足しなかった）場合は「計算の誤り」に該当しない。ただし、この法理は平成23年改正前の事案であることに注意。

2-b. その他の裁決例

裁決事例集No.65-34（平15.5.20裁決）― やむを得ない事情の否認

事案：マレーシア子会社からのロイヤリティに係る源泉税について、外国税額控除の記載・添付を全くしなかった。外国法人税が課されていること自体を知らなかったと主張。

判断：「やむを得ない事情」とは本人の責めに帰すことのできない客観的事情をいう。ロイヤリティ契約書に源泉税条項があり、代表者が契約内容を承知していた以上、通常要求される確認作業で判明し得たものであり、やむを得ない事情は認められない。

裁決事例集No.74-09（平19.10.9裁決）― 手続要件不備による否認

事案：米国不動産の譲渡所得について、米国での確定申告時期と日本の申告時期のずれから、外国税額控除の適用手続が適切にされなかった。更正の請求で事後的に外国税額控除の適用を主張。

判断：「外国税額控除の制度は、申告期限内であるか否かを問わず、最初に提出する納税申告書に所定の記載等をすることを要件としている」。所得税法95条2項・5項が控除方法を明確に規定しており、この規定に従った申告を行っていなかった以上、更正の請求は認められない。修正申告での外国税額控除の適用を認めないことへの不服は「立法論」であるとした。

※この裁決も平成23年改正前のもの。

3. 平成23年改正（当初申告要件廃止）の射程

3-a. 改正の概要

平成23年12月税制改正の骨子

  1. 更正の請求期間の延長：法定申告期限から1年 → 5年
  2. 当初申告要件の廃止：外国税額控除を含む多くの税額控除について、確定申告書（当初申告）に記載がなくても、修正申告・更正の請求で事後的に適用可能に
  3. 記載金額限度要件の廃止：「当該金額として記載された金額を限度とする」という制限の廃止

適用時期：平成23年12月2日以後に法定申告期限が到来する国税に適用。

改正による実務上の変更点

  • 損金算入→税額控除への変更：当初申告で外国法人税を損金経理した場合でも、更正の請求で税額控除方式に変更可能
  • FTCの事後適用：当初申告でFTCを全く申告していなかった場合でも、更正の請求でFTCを新規に主張可能
  • FTCの増額：当初申告でFTCの金額に誤りがあった場合、更正の請求で控除額の増額が可能

3-b. 法人税法69条25項の趣旨と構造

法人税法69条25項（現行法） 第1項の規定は、確定申告書、修正申告書又は更正請求書に第1項の規定による控除を受けるべき金額及びその計算に関する明細を記載した書類並びに控除対象外国法人税の額の計算に関する明細その他の財務省令で定める事項を記載した書類の添付があり、かつ、控除対象外国法人税の額を課されたことを証する書類その他の財務省令で定める書類を保存している場合に限り、適用する。

重要な分析：69条25項の文言の意義

改正前の旧16項は「確定申告書に……記載があり、かつ……書類の添付がある場合に限り」と規定しており、確定申告書のみを対象としていた。

現行25項は「確定申告書、修正申告書又は更正請求書」と規定することで、以下の場面を包摂する。

場面25項の対応
当初申告でFTCを申告していなかった場合に、更正の請求で新規にFTCを主張 更正請求書に明細書類を添付し、証拠書類を保存していれば適用可
当初申告でFTCを申告済みだが、修正申告で控除額を変更 修正申告書に明細書類を添付すれば適用可
当初申告でFTCを申告済みだが書類添付が不完全な場合に、更正の請求で補完 更正請求書に不足書類を添付すれば適用可（下記分析参照）

書類添付が不完全な場合の更正の請求の可否

69条25項が「確定申告書、修正申告書又は更正請求書」と並列で規定していることから、以下のように解される。

  1. 25項の趣旨は、FTCの適用に際していずれかの段階で必要な書類の添付・保存があることを求めるもの
  2. 当初申告で書類添付が不完全であっても、更正請求書に必要書類を添付し、証拠書類を保存していれば、25項の要件を充足する
  3. ただし、これは平成23年改正後の法文の解釈であり、改正前の事案（旧16項が「確定申告書」のみを対象としていた時代）には適用されない

なお、69条28項は「やむを得ない事情」がある場合には、書類の保存がなくても適用できる旨を規定している（添付要件ではなく保存要件の緩和である点に注意）。

4. 更正の請求の「税額が過大」要件

核心的問題：既にFTCを申告している場合の「税額過大」の判断

通則法23条1項1号は「納付すべき税額が過大であるとき」を要件とする。この「過大」の判定基準が問題となる。

場面税額過大の該当性理由
FTCを全く申告しておらず、本来控除すべき額がある 該当する 法令に従った計算では控除後の税額が少なくなるため過大
FTCを申告済みだが計算誤りにより控除額が過少 該当する 正当に計算した控除額はより大きく、結果として納付税額が過大（No.77-21で認容）
FTCを申告済みで計算は正しいが、添付書類が不完全 微妙（改正前後で異なる） 改正前：書類不備→FTC適用不可→税額は「適正」→過大でない
改正後：更正請求書で書類補完可能→本来の控除額との差額は過大に該当し得る
FTCを申告済みで控除額も適正（又は過少申告ではない） 不該当 税額が過大でない以上、更正の請求の余地なし

「過大」の判定基準

「過大」とは、法律の規定に従い正当に計算した場合の税額と比較して過大であることを意味する。

  • FTCの計算に誤りがあった場合：正当に計算したFTC額は申告額より大きい → 納付税額は「過大」
  • FTCの適用自体を選択しなかった場合（改正後）：法令に従えばFTCを適用した税額が「正当」→ 適用しなかった税額は「過大」
  • FTCの適用自体を選択しなかった場合（改正前）：適用要件（記載・添付）を充足しない以上、FTCの不適用が「法律の規定に従った」結果 → 税額は「適正」であり「過大」ではない

つまり、改正前は手続要件を充足していないことが「適正」な申告とされ、改正後は実体的に本来控除されるべき額との比較で「過大」が判定されるという構造転換が生じた。

5. 総合分析と実務上の留意点

分析のまとめ

（1）更正の請求が認められる典型的場面（現行法下）

  • 当初申告でFTCを適用したが、控除額の計算に誤りがあった場合 → 更正の請求で増額可能（No.77-21の法理が引き続き妥当）
  • 当初申告でFTCを全く適用しなかった場合 → 更正請求書に必要書類を添付し、証拠書類を保存していれば、更正の請求でFTCの新規適用が可能（平成23年改正による当初申告要件廃止）
  • 当初申告で外国法人税を損金算入していた場合 → 更正の請求で税額控除方式に変更可能（当初申告要件廃止の効果）

（2）添付書類が不完全な場合の更正の請求

  • 69条25項が「確定申告書、修正申告書又は更正請求書」に添付と規定 → 更正請求書に不足書類を添付することで要件充足可能
  • ただし、「保存」要件（控除対象外国法人税の額を課されたことを証する書類の保存）は別途充足する必要あり
  • 書類の保存がない場合は、69条28項の「やむを得ない事情」の認定が必要

（3）実務上のリスクポイント

  • 更正の請求の5年間の期限に留意（法定申告期限から起算）
  • 「やむを得ない事情」は厳格に解釈される傾向（単なる失念・不注意は該当しない）
  • 平成23年改正前の事業年度については旧法が適用 → 当初申告要件が存続
  • 69条15項～17項の「当初申告税額控除額」による調整規定にも留意が必要（平成29年改正で増額更正時の調整規定が整備）
