1. GloBEルールの全体構造

GloBE（Global Anti-Base Erosion）ルールは、OECD/G20 BEPS包摂的枠組みにおけるPillar Two（第2の柱）の中核をなす制度である。2021年10月のBEPS包摂的枠組み（当時136か国・地域が合意）において最終合意され、同年12月にGloBEモデルルール、2022年3月にコメンタリーが公表された。

基本原則

  • 年間連結売上高7.5億ユーロ（約1,275億円）以上の多国籍企業グループを対象
  • 各国・地域ごとに算定した実効税率（ETR）が最低税率15%を下回る場合、差額分をトップアップ税として追加課税
  • 課税標準は連結財務諸表（IFRS又は適格GAAP）ベースの税引前利益から算出
  • 実質ベースの所得除外（Substance-based Income Exclusion）：有形資産簿価の5%＋支払給与の5%を控除（導入当初は経過措置で高い率）

ルールの適用順序

適用優先順位： QDMTT（国内ミニマム課税）→ IIR（所得合算ルール）→ UTPR（軽課税所得ルール）
QDMTTが適用された場合、当該分についてIIR・UTPRの課税は行われない。STTRは租税条約ベースの別個のルールとして独立に機能する。

GloBEルールは4つの構成規則からなる。

ルール性質法的根拠機能
IIR一次ルール（Primary）国内法親会社所在国でトップアップ税を課税
UTPR補完ルール（Backstop）国内法IIRで捕捉されない残余分を子会社所在国で配分課税
QDMTT防御ルール（Defensive）国内法自国のETRを15%まで引き上げ、他国からの課税を防止
STTR条約ルール租税条約特定のグループ内支払に対する源泉地国の課税権を確保

2. IIR（Income Inclusion Rule / 所得合算ルール）

2-1. 基本構造

IIRは、CFC税制と同様の原理に基づくが、決定的な違いがある。CFC税制が外国子会社の個別の所得を合算するのに対し、IIRは国別の実効税率に基づいてトップアップ税額を算出し、それを各構成会社に配賦する点である（GloBEモデルルール Art.2.1-2.5、コメンタリーArt.10パラ8）。

2-2. トップアップ税額の計算

  1. 国別ETRの算定： ETR = 調整後対象租税額 / GloBE所得（国別集計）
  2. トップアップ税率： 15% - 当該国のETR（ETR < 15%の場合のみ発生）
  3. 超過利益の算出： GloBE所得 - 実質ベースの所得除外額（SBIE）
  4. トップアップ税額： 超過利益 × トップアップ税率 + 追加当期トップアップ税額

2-3. 適用順序（Top-down approach）

IIRは「トップダウン・アプローチ」により適用される。すなわち、最終親会社（UPE）がまずIIRを適用する。UPEの所在地国がIIRを導入していない場合は、所有構造を下って、中間親会社（IPE）がIIRを適用する。この適用順序により、複数の親会社によるIIRの重複適用が回避される。

2-4. CFC税制との関係

IIRとCFC税制は併存する（GloBEモデルルール Art.10.1 コメンタリー パラ8）。CFC税制はGloBEルールに先行して適用され、CFC税制に基づく親会社への課税額は、子会社に「プッシュダウン」（配賦）され、ETR計算上考慮される（Art.4.3.2(c)コメンタリー パラ58）。ただし、受動的所得（パッシブ所得）に係るCFC税の配賦は一部制限される（Art.4.3.3）。
【典拠】岡直樹「ピラー２とCFC税制」（2022年6月17日）17頁：「国内法のCFC税制がGloBEルール（QDMTT→IIR→UTPR）に優先して適用される」。Art.4.3.2(c) コメンタリー パラ45参照。

3. UTPR（Undertaxed Profits Rule / 軽課税所得ルール）

3-1. バックアップメカニズム

UTPRは、IIRの「安全装置」（backstop）として機能する二次的ルールである。トップアップ税額のうちIIRの対象とされていない部分（残余のトップアップ税額）について、子会社等の所在地国においてその子会社等に対して課税を行う制度である。

典型的な適用場面は以下の通りである。

  • 最終親会社の所在地国がIIRを導入していない場合
  • 最終親会社自体が軽課税国に所在し、自国のトップアップ税額はIIRの対象外となる場合
  • 所有構造上、IIRでは捕捉できない構成会社がある場合

3-2. 配分計算式

UTPRに基づくトップアップ税額（グループ国際最低課税残余額）は、UTPR導入国間で以下の配分式に基づき按分される。

UTPR配分割合 =
（自国所在の構成会社の従業員数 / UTPR導入国全体の構成会社の従業員数）× 50%
＋
（自国所在の構成会社の有形資産額 / UTPR導入国全体の構成会社の有形資産額）× 50%

日本法では、法人税法第82条の11に規定される「国際最低課税残余額」の計算において、この配分式が採用されている（法82の11①②、令155の59①～⑤）。

3-3. 日本法上の実装

令和7年度税制改正により、「各対象会計年度の国際最低課税残余額に対する法人税」として法制化された。適用開始は令和8年（2026年）4月1日以後に開始する対象会計年度から。税率は100分の90.7（法82の13）。

4. QDMTT（Qualified Domestic Minimum Top-up Tax / 国内ミニマム課税）

4-1. 制度趣旨と防御機能

QDMTTは、自国での税負担が基準税率15%に至るまで課税を行う制度であり、IIRやUTPRによる他国からの課税を防止する防御的機能を持つ。自国がQDMTTを導入し、自国分のトップアップ税額を自国で徴収すれば、当該分についてIIR・UTPRの適用は不要となる。

4-2. IIR・UTPRに対する優先適用

GloBEルールの適用順序上、QDMTTは IIR・UTPR に優先する。QDMTTにより課された税額は、IIRの計算上、当該国のトップアップ税額から控除される。したがって、ある国がQDMTTを導入していれば、当該国に係るトップアップ税額は原則として生じない（当該国のQDMTTで既に徴収済みのため）。

4-3. 国内実効税率の計算

日本法における国内最低課税額の計算は以下の構造をとる（法82の19②）。

  1. 国内グループ純所得の金額 = 日本所在の全構成会社の個別計算所得金額合計 - 個別計算損失金額合計
  2. 実質ベースの所得除外額 = 給与等費用額 × 5% + 有形固定資産等額 × 5%（経過措置あり）
  3. 国内実効税率 = 国内調整後対象租税額合計 / 国内グループ純所得の金額
  4. 当期グループ国内最低課税額 = （国内グループ純所得 - 所得除外額）×（15% - 国内実効税率）

QDMTTの税率は100分の75.3（法82の21）。適用開始は令和8年（2026年）4月1日以後に開始する対象会計年度から。

4-4. 適用免除基準

  • デミニマス： 収入金額が1,000万ユーロ未満かつ税引前利益が100万ユーロ未満の場合、トップアップ税額を0とすることを選択可（法82の19⑧）
  • 移行期間CbCRセーフハーバー： 令和8年4月1日〜同年12月31日開始の対象会計年度（令和10年6月30日まで終了分に限る）について、一定要件充足で0選択可（令和7年改正法附則18①②）
  • 国際事業活動の初期段階： 構成会社の所在地国数6以下、かつ特定所在地国以外の有形資産合計5,000万ユーロ以下の場合、5年間免除（法82の19⑭）

5. STTR（Subject to Tax Rule / 租税条約の特典否認ルール）

5-1. 租税条約ベースのルール

STTRは、GloBEの国内法ベースの3ルール（IIR/UTPR/QDMTT）とは異なり、租税条約に基づくルールである。関連当事者間の支払い（利子・使用料等の控除可能支払い）が受取人の管轄区域において名目税率9%未満で課税されている場合に、源泉地国が追加の源泉課税を行えるようにする。

5-2. 適用範囲

項目内容
適用対象支払利子、使用料、賃料、サービス料、保険料、配当等のグループ内控除可能支払い
基準税率9%（名目税率ベース。GloBEの15%とは別の基準）
追加課税額9% - 受取人所在国の適用税率 = 源泉地国が追加課税可能な税率
主な対象国開発途上国（1人当たりGNI 12,535ドル以下の国）が要請した場合に二国間条約に組み込む
多国間協定2023年10月にSTTR MLI（多国間協定）公表。2024年11月時点で9か国が署名

5-3. IIR/UTPRとの関係

STTRに基づき課された追加税は、IIR・UTPRのETR計算上考慮される（GloBEモデルルール Art.4.3.2(c)）。したがって、STTRによる課税が先行し、その分ETRが上昇するため、IIR/UTPRのトップアップ税額は減少する。

菊谷正人「最低法人税率の全世界的導入」イノベーション・マネジメント No.19: 「STTR（課税対象規則）は、他の契約管轄区域の低い名目税率を利用するグループ内支払いに関連するBEPS構造によってもたらされる源泉地国のリスクを対象とする条約ベースの規則である。」

6-3. CFC税制との適用調整規定

制度的位置づけ：併存関係

令和5年6月の政府税制調査会答申は、以下の整理を示している。

「既存の外国子会社合算税制は、経済的な実体の乏しい子会社等を用いた租税回避に対処することを目的とするのに対して、『第2の柱』は、各国共通の最低税率の導入により法人税引下げ競争に歯止めをかけることを目的とするものであり、両者は目的を異にする別個の仕組み」
「国際的なルールにおいても、CFC税制は『第2の柱』と併存するものと整理されている」

令和7年度改正における外国子会社合算税制の見直し

グローバル・ミニマム課税の導入に伴う追加的事務負担への配慮として、以下の見直しが行われた（措法66の6、66の7）。

  • 課税対象金額の算入時期の見直し： 外国関係会社の事業年度終了の日の翌日から4月（改正前：2月）を経過する日を含む内国法人の事業年度に算入（措法66の6①⑥⑧）
  • 控除対象からの除外： 国際最低課税残余額に対する法人税の額及びその法人税に係る地方法人税の額は、CFC税制における控除対象所得税額等相当額の範囲から除外（措法66の7④）
  • トリガー税率の調整： 令和5年度改正で適用免除税率の閾値を30%未満から27%未満に変更済み

博士論文提案（条文骨子案7）との関係

博士論文第11章で提案された「条文骨子案7：グローバル・ミニマム課税との調整」（租税特別措置法第66条の6の3案）は、以下の2点を規定する。

  1. CFC税制のトリガー税率判定において、トップアップ税の額を外国法人税の額に含める
  2. グローバル・ミニマム課税により既に課税された金額についてはCFC税制を適用しない

また「条文骨子案8」では、CFC税制→逆ハイブリッド事業体特例→損金算入配当益金算入→グローバル・ミニマム課税の適用順序を明確化する提案がなされている。

8. 博士論文における論点整理

8-1. 「閾値下に残る制度的空白」（ch07テーマ）

Pillar Twoは年間連結売上高7.5億ユーロ以上の多国籍企業のみを対象とする。この閾値を下回る企業は引き続きCFC税制のみによって規律される。この「閾値下の空白」が、CFC税制の独自の存在意義を示す。

8-2. 岡直樹氏の問題提起（2022年）

岡論文は以下の核心的問題を提起する。

  • GloBEルールにより「租税回避のしにくい世界」が出現する以上、CFC税制のBEPS対抗的要素は「大胆な合理化」を検討しうる
  • CFC税制（29.7%で課税）とIIR（15%トップアップ）の二重適用のコスト問題
  • 軽課税国がQDMTTを導入すれば日本はIIR課税できなくなるが、CFC課税はQDMTTに優先する
  • 「経済活動基準のうちBEPS対抗措置といいうる部分（GloBEルールがカバーする部分）に着目してはどうか」

8-3. METI研究会の論点（令和6年11月）

経済産業省「日本企業の海外展開動向を踏まえた国際課税制度のあり方に関する研究会」は以下を論点に掲げている。

  • グローバル・ミニマム課税導入により軽課税国への移転インセンティブが低下する中、CFC税制の役割はどう変化するか
  • グローバル・ミニマム課税の対象企業について、CFC税制の課税対象範囲を見直すことは妥当か
  • 適用免除税率の閾値引下げによるCFC税制の対象絞り込みは妥当か
  • 両制度の重複によるコンプライアンスコストの増大への対応

8-4. 博士論文の位置づけ

博士論文ch07「Pillar Twoとのすみ分け：閾値下に残る制度的空白」は、鍋谷論文の脚注156-174（OECD CFC勧告パラ94関連）を中心典拠とし、CFC税制がPillar Twoとは異なる独自の機能を果たす領域を析出する章として設計されている。これは第11章の条文骨子案7（グローバル・ミニマム課税との調整）及び条文骨子案8（適用順序の明確化）の提案と直結する。
