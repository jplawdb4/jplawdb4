まず結論

日中配当のみなし外国税額控除の法的な芯は、条約23条3(a)が単独で国内法を動かすのではなく、23条2項(a)の「日本の国内法に従ったFTC」 に乗って、69条1項の「控除対象外国法人税額」 の計算素材へ入り込む、という接続にある。したがって、条約側だけ読んで終わりではなく、国内法側の 69条25項 と 実施特例法施行省令10条 まで見る必要がある。

実務上の現行申告導線はおおむね B06(四) → B06(二の二) → B06(二) → B01/B04 である。古い裁決では旧別表六(五)系の番号が見えることがあるが、現行フォームにそのまま機械的対応させない方が安全である。

上段が条約レイヤー、中段が国内法レイヤー、下段が現行別表レイヤーである。中央の矢印だけ追うと、23条3(a)の「みなし税率」 がどこで 法69条1項 に接続し、どこで 添付・証明書類 の要求へ変わるかが掴める。

最短で押さえる3点
  • 中国で実際に10%しか引かれていなくても、23条3(a) は日本側FTC計算上 10%又は20% を「支払ったものとみなす」。
  • その条約上のみなし税額は、69条1項 の FTC 本体へ落ちるが、適用の実務上は 69条25項 と 実施特例法施行省令10条 がかなり重い。
  • 現行フォームでは、みなし部分を外形化する起点はおおむね B06(四) であり、その後 B06(二の二)、B06(二) を経て税額控除に反映する。

2. 読み筋

10条2項

中国側の配当課税を認めつつ、受益者要件を満たす場合の税率を 10%以下 に抑える。ここは「実際課税」の上限であって、日本側のみなし率そのものではない。

23条2項(a)

日本は「日本国外で納付される租税を日本で控除する国内法」に従って中国税を控除する。つまり、条約だけで完結せず、国内法 69条 の受皿が必要になる。

23条3項(a)

23条2項(a)の適用上、中国税を 合弁企業配当なら10%、その他の配当なら20% 支払ったものとみなす。ここが配当のタックス・スペアリングの主位根拠である。

69条1項

みなし税額を含む中国税を 控除対象外国法人税額 として、日本法人の法人税額から控除する本則。条約上の「みなし」が実際に計算へ入るのはここ。

69条25項・規29条の4

一般のFTCについて、明細書添付と証明書類保存を要求する。原則として、明細書記載額が控除額の上限になる。

実施特例法施行省令10条

みなし外国税額がある場合には、一般FTC書類に加えて みなし税額の計算明細 と みなし税額を証明する書類 を添付書類群に組み込む。

23条2項(b)との関係

日本法人が中国支払法人の議決権株式又は発行済株式を 25%以上 保有していれば、23条2項(b) による間接外国税額控除の問題も併走し得る。ただし、これは 23条3(a) の固定みなし率と同じルートではないため、図では脇分岐として置いている。

3. 現行別表のつながり

段階 現行別表の役割 このテーマでの意味
B06(四) 国別・税目別に納付分とみなし分を個別明細化 日中条約23条3(a)を使うなら、ここで中国配当・条約根拠・みなし税額を外形化するのが出発点になる。
B06(二の二) 当期の控除対象外国法人税額を総括 B06(四)の納付分・みなし分を集約して、当期に69条1項へ流す金額をまとめる。
B06(二) 69条1項の控除限度額・控除額を計算 ここで初めて、みなし税額を含む中国税が法人税の FTC 本体計算に入る。
B01 / B04 税額控除・加算の最終転記先 FTC が法人税申告書全体の税額計算へ反映する出口になる。
古い裁決番号とのズレ

古い裁決・審判所事例では、みなし外国税額控除に関して 別表六(五) など旧番号のフォームが現れる。現行フォームへ落とすときは、番号の一致だけで追わず、「個別明細 → 総括 → 本体計算」 という機能対応で読む方が安全である。
