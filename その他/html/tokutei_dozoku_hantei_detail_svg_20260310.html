まず結論

特定同族会社の判定は、単に 「筆頭グループが50%超か」 を見るだけでは終わらない。正確には、法67条2項 + 令139条の7 でまず被支配会社に当たるかを見て、その判定基礎に 被支配会社でない法人 が混じれば 法67条1項 に従って除外再判定し、さらに 1億円以下なら 法66条5項2号〜5号又は6項 に当たるかを確認し、最後に清算中でないことまで確かめて、はじめて法67条1項の「特定同族会社」に到達する。

したがって、実務上の核心は四つある。第一に、期末現在 で判定すること。第二に、株式数だけでなく議決権・社員数 でも50%超が成立し得ること。第三に、非被支配会社法人株主等の除外再判定 を飛ばさないこと。第四に、1億円以下だから自動的に対象外になるわけではない こと。大法人の完全子法人等や大通算法人は、この最後の分岐で再び特定同族会社側へ戻ってくる。

既存ページとの分担

既存の判定表 は俯瞰用、本頁は精査用である。入口をざっと掴むなら前者、相互持合い・同意議決権・1億円以下例外まで安全に読み切るなら本頁、という使い分けを想定している。

1. 詳細SVG

中央の本流が条文の順番、左列が定義・入力補正、右列が読み違えやすい補助論点、下段が除外再判定のミニ設例である。実務では、中央を上から下へ辿り、迷った論点だけ左右と下段へ戻ると速い。

最短で押さえる3点
  • 特定同族会社判定は 上位1グループ が50%超かで出発する。上位3グループの同族会社判定と混同しない。
  • 被支配会社でない法人株主等がいるときは、除外してもう一度50%超か を見る。ここを省くと誤判定しやすい。
  • 1億円以下でも、大法人完全子法人等や大通算法人なら特定同族会社側へ戻る。小規模 = 自動的に対象外 ではない。

2. 条文の骨格

詳細判定フローを条文単位で切り出すと、次の5ブロックに整理できる。

法2条十

同族会社の基礎規定。上位3株主グループで50%超かを測る土台であり、別表二の同族会社判定割合につながる。

法67条2項

特定同族会社判定の核。被支配会社とは、一人の株主等と特殊関係個人・法人が、株式等総数等の50%超を持つ会社その他政令所定の場合をいう。

令139条の7

特殊関係個人・法人の範囲、支配の3ルート、同意議決権の吸収を定める。特定同族会社判定を実際に回すための作動規定である。

法67条1項

非被支配会社法人の除外再判定、1億円以下法人への限定、清算中除外を一つに抱える。特定同族会社かどうかの最後の絞り込みはここで行う。

法67条8項

判定時点は各事業年度終了の時の現況であると固定する。期中の株主構成ではなく、期末の事実で決まる。

法66条5項・6項

大法人完全子法人等や大通算法人など、中小軽減税率の適用除外法人群を定める。法67条1項は1億円以下法人についてこの群を参照する。

通達16-1-3の意味

相互持合いがあるとき、非被支配会社法人の除外再判定を片側だけで終えてはいけない。通達16-1-3は、当該法人と相手法人の双方について除外後も被支配会社かを見よと補足している。条文のかっこ書きは、相互持合いに入るほど重要になる。

3. ミニ設例

除外再判定のイメージが掴みにくいので、あえてそこだけ二つの小設例に切り出す。

ケース 初回判定 除外再判定 結論
A A本人32% + 配偶者12% + A支配法人B10% + 非被支配会社法人C8% = 62% Cを外して 54% 除外後も50%超なので、他要件も満たせば特定同族会社へ進む
B A本人29% + 配偶者11% + 非被支配会社法人C12% = 52% Cを外して 40% 50%以下に落ちるため、特定同族会社にはならない
設例の読み筋

両ケースとも初回は50%超だが、法67条1項の除外再判定を通すと結論が分かれる。したがって、B02 17欄だけで「特定同族会社」と即断せず、その50%超に非被支配会社法人が混ざっていないか を必ず点検する必要がある。

4. 別表二の読み方

別表二は条文判定をフォームに落としたものであり、実務上は次の欄を軸に読むと混乱しにくい。

欄 意味 このページでの位置づけ
10欄 同族会社の判定割合 上位3グループで50%超か。法2条十の基礎判定。
17欄 特定同族会社の判定割合 上位1グループで50%超か。法67条2項 + 令139条の7 の入口。
被支配会社でない法人株主等欄 非被支配会社法人の抽出 法67条1項の除外再判定へ進むための材料。
18欄 判定結果 最終的に「特定同族会社 / 同族会社 / 非同族会社」を表示する出口。
別表二だけで完結しない理由

17欄が50%超でも、清算中であったり、1億円以下で法66条5項2号〜5号・6項に当たらなかったり、除外再判定で50%以下に落ちたりすれば、法67条1項の「特定同族会社」にはならない。別表二は強いシグナルだが、最終結論は条文側で閉じる。
