みなし外国税額控除・省令10条未添付と更正の請求

本稿は、「法人税法69条25項にいう添付要件、特に実施特例法施行省令10条書類の欠缺は、更正の請求によって原則として補完できない」という厳格否定説を、条文構造、旧法との比較、裁決・判例、立法経緯、反対説への反駁材料まで含めて組み上げるための資料総覧である。

作成日: 2026-03-12 / 対象論点: みなし外国税額控除・更正の請求・省令10条未添付 / 研究方針: 条文から組み、77_22を主軸に据え、77_21・11242は射程限定材料として扱う。

結論先取り

  1. 法人税法69条25項 は、外国税額控除の適用につき、申告書等への明細書の添付と証拠書類の保存を 「場合に限り」 の文言で要件化している。
  2. 同28項 は、そのうち 保存欠缺だけ を「やむを得ない事情」により救済する。添付欠缺について同様の救済規定はない。
  3. 実施特例法施行省令10条 は、みなし外国税額控除に特有の計算明細と証明書類を、69条25項の添付書類の内容として具体化する。
  4. 通則法23条1項1号 による更正の請求は、法令違反又は計算誤りを要する。25項要件未充足のまま控除を認めなかった申告は、厳格否定説からはむしろ法令適合的である。
  5. したがって、省令10条書類の欠缺を更正の請求で後補することは、単なる計算是正ではなく、新たな制度適用要件の事後的充足 又は みなし部分の適用範囲の新規追加 と評価されやすい。

この論点では、「更正請求書が条文に出てくること」と「添付欠缺が治癒できること」は別問題 である。厳格否定説の芯は、25項の適用要件構造と28項の限定救済構造を分けて読む点にある。

1. 論点の定式化

想定事例は次のとおりである。内国法人がみなし外国税額控除の適用を念頭に別表6系列を提出したが、租税条約等実施特例法施行省令10条 所定の書類、すなわち みなし外国税額の計算明細 と みなし外国税額を証明する書類 を添付していなかった。その後、税務調査でみなし部分が否認され、更正がされた。ここで、更正後の 通則法23条 に基づく更正の請求により、当該省令10条書類を後出ししてみなし部分の控除を回復できるかが問題となる。

肯定説は、現行 法人税法69条25項 が「確定申告書、修正申告書又は更正請求書」と書いている以上、更正請求書への後添付も制度上予定されているという文理解釈に立つ。これに対し厳格否定説は、更正請求書という提出媒体が開かれていること と、当初に欠けていた添付要件が23条上の計算誤りとして後から治癒できること とは別だとみる。

厳格否定説の基本命題 はこう要約できる。省令10条書類の欠缺は、単なる証拠の不足ではなく、69条25項所定の適用要件の未充足である。したがって、その欠缺を更正の請求で補完することは、23条1項1号のいう計算是正ではなく、新規に適用要件を充足し直す試みであるから、原則として許されない。

2. 現行法の条文構造

2.1 法人税法69条25項: 添付と保存を並べて「場合に限り」

法人税法69条25項 の現行文言は、外国税額控除を受けるためには、申告書等に明細書の添付があり、かつ、証拠書類を保存している場合に限り、適用する と定める。そして後段で、税額の計算の基礎となる金額は、原則として 明細書に記載された金額を限度 とする。

条文要素 意味 厳格否定説での使い方
申告書等 確定申告書・修正申告書・更正請求書の三媒体を列挙 提出媒体の拡張にすぎず、要件の消滅ではない
明細書の添付があり 添付は効力発生の一部 後に添付が欠けていたことが判明した場合、まず25項の適用成否が問われる
保存している場合に限り 証拠書類保存も独立要件 28項がこの保存だけを救済していることが対比上重要
明細書記載額を限度 申告書等で外形化された範囲を固定 後から範囲自体を広げることを嫌う規律と読める

2.2 法人税法69条28項: 救済は保存欠缺だけ

69条28項 は、前三項に規定する財務省令で定める書類の保存がない場合 であっても、「やむを得ない事情」があれば税務署長は適用できるとする。ここで条文は、添付 の欠缺ではなく、保存 の欠缺だけを救済している。

この分業構造は、厳格否定説の最重要根拠である。もし添付欠缺も保存欠缺と同じように後から治癒できるなら、28項が保存だけを切り出して救済規定を置く意義が大きく減る。反対説は、この対比を正面から説明しなければならない。

2.3 実施特例法施行省令10条: みなし部分の添付内容を具体化

実施特例法施行省令10条 は、外国税額の控除を受けようとする場合に、その外国税額のうちに みなし外国税額 があるときは、69条25項等で添付すべき書類の中に、みなし外国税額の計算明細 と 当該みなし外国税額を証明する書類 を含むとする。したがって、省令10条書類は独立の参考資料ではなく、69条25項の「財務省令で定める事項を記載した書類」や「財務省令で定める書類」の中身を、みなし部分について具体化したものと読める。

2.4 通則法23条1項1号: 更正の請求は計算誤りと法令違反に限る

国税通則法23条1項1号 は、更正の請求の通常ルートを、申告書に記載した課税標準又は税額の計算が法令に従っていなかったこと 又は 当該計算に誤りがあったこと により税額が過大であるときに限る。厳格否定説では、25項の適用要件が満たされない以上、みなし外国税額控除を認めずに申告したことは 法令違反ではなく、むしろ法令適合的 とされる。したがって、更正の請求は23条の入口で止まる。

2.5 条文構造から導かれる帰結

  1. 69条25項は、添付と保存を併せて控除適用の要件にしている。
  2. 69条28項は、そのうち保存だけを宥恕する。
  3. 省令10条は、みなし部分に特有の添付資料を25項に組み込む。
  4. 23条は、要件未充足の後追い選択のための一般救済条項ではない。

この四点を積み上げると、厳格否定説は、単に「気持ちとして厳しく読む」のではなく、現行条文の役割分担を忠実に読む立場 と位置づけられる。

3. 旧法から現行法への変化

3.1 旧69条16項・18項の構造

旧法では、77_21・77_22が適用した旧 法人税法69条16項 が、確定申告書への記載及び書類添付を要し、かつ記載額限度のルールを置いていた。他方、旧 同18項 は、記載又は書類添付がない場合でも、「やむを得ない事情」があれば救済し得る構造をとっていた。

この旧法は、当初申告要件が強く、ただし記載・添付について宥恕余地も明文で置いていた と要約できる。

3.2 平成23年度改正の意味

平成23年度改正後の現行25項は、修正申告書・更正請求書 まで媒体を広げた。これにより、旧来の意味での「確定申告書だけに閉じる」当初申告要件は緩和された。その点は国税庁の「当初申告要件が廃止された措置」一覧でも、外国税額控除が掲げられていることで確認できる。

重要なのは、改正で広がったのは提出媒体であって、添付欠缺を自由に後治癒できる制度に変わったわけではない という点である。むしろ現行法は、旧18項のような「記載又は添付」全体に対する宥恕を維持せず、28項で保存欠缺だけを救済対象として残した。ここから、立法者が添付と保存を意図的に峻別したと読む余地が強まる。

4. 厳格否定説を支える裁決・判例群

4.1 中核: 77_22

77_22 は本論点の主砲である。みなし外国税額控除について、別表一(一)、別表六(五)、別表六(五の三)、証明書類が欠けた状態で更正の請求がされた事案において、審判所は、記載及び添付の要件を充足しない以上、やむを得ない事情がない限り適用不可 とし、さらにその結果として、通則法23条1項1号にも該当しない とした。

厳格否定説における使い方は明確である。省令10条書類の欠缺を、単なる立証不足ではなく適用要件不充足と位置づけ、そのまま23条の入口否定につなげる 点で、事実関係も論理も最も近い。

4.2 強い一般理論: 11812

11812 は所得税の外国税額控除事案だが、論理構成が非常に強い。すなわち、一定の手続要件を具備していない以上、外国税額控除を適用しない申告は 国税法規に従っている のであり、そのような申告については通則法23条1項1号の更正の請求要件を満たさない、とした。

みなし外国税額控除は条約を背景に持つため複雑だが、要件未充足の申告が違法ではなく適法である という骨格は、厳格否定説の一般理論としてそのまま転用しやすい。

4.3 外形化責任: 74_09

74_09 は、外国税額控除を当然適用の制度ではなく、確定申告書に自ら意思内容を明確に示すことを要する制度 とみる方向を補強する。みなし外国税額控除でも、単に外国税が背景にあるというだけでは足りず、どの税額のどの成分を控除対象とするか を申告書等で外形化する必要がある。

4.4 宥恕規定の狭さ: 65_34

65_34 は、旧法人税FTCに関する「やむを得ない事情」を、本人の責めに帰すことのできない客観的事情 に限定して読む。税理士の失念、社内の確認不足、資料収集の遅れのような人的・主観的事情は救済されにくい。

現行法では、そもそも28項が保存欠缺しか救済していないため、厳格否定説は、旧法よりもむしろ宥恕余地は狭い と反論できる。

4.5 現行法側の補強: 13531・13633

13531 や 13633 は、現行法の外国税額控除でも、申告書等に添付された明細によって対象税額の範囲を外形化していない以上、後から当然連動や別資料提出で範囲を拡張することを許さない方向を示す。これらはみなし外国税額控除そのものではないが、現行69条でも外形化責任がなお強く残る ことを示す現在法側の重要材料である。

4.6 他税目の類推: 69_28, 12879, 13233

69_28

登録免許税で、特例適用のために申請時証明書類添付を要し、後日提出による還付請求を認めない方向を示す。税目は違っても、減免の入口要件としての添付書類は後治癒不可になり得る という一般法理の補強になる。

12879 / 13233

いずれも税額控除制度における当初申告要件・適用額の制限を扱い、後から追加的に適用範囲を拡張する更正の請求等は許されない という発想を明確にする。とくに、条文に更正請求書が出てくること自体から直ちに後治癒可とはいえない 点で、厳格否定説にとって使いやすい。

4.7 射程限定が必要な陽性例: 77_21 と 11242

5. 反対説の整理と反駁材料

5.1 「更正請求書を明文で読むのだから、後添付できる」論

反対説 は、69条25項と省令10条がいずれも更正請求書を明文で読むのだから、更正の請求段階での書類補完は制度上当然に許容されていると主張する。

反駁 は次の二段でよい。

  1. 条文が広げたのは 提出媒体 であって、23条1項1号に入るための理由 までは無条件に広げていない。
  2. 69条28項が保存欠缺だけを救済している以上、添付欠缺の後治癒を広く認めると条文構造が崩れる。

5.2 「別表6がある以上、選択は済んでいる」論

反対説 は、別表6に外国税額控除関連の記載があれば、少なくとも制度選択の意思表示は済んでおり、省令10条書類の欠缺は立証不足にとどまるとみる。

反駁 のポイントは、「選択の開始」と「選択の完了」を分けること にある。みなし部分は、通常FTCの付随成分ではなく、条約要件、軽減・免除の事実、みなし税額の計算という固有の前提を持つ。77_22が示すのは、通常部分を選択していても、みなし部分まで当然に選択済みとは言えないということである。

5.3 「省令10条書類は立証資料にすぎない」論

反対説 は、みなし外国税額控除の実体的権利自体は条約と69条により存在し、省令10条書類はそれを証明する資料にすぎないから、23条上の更正の請求で後から補えるという。

反駁 は、69条25項と28項の構造に戻るのが最も強い。もし省令10条書類が単なる証拠提出の問題にすぎないなら、25項で「場合に限り」とし、28項で保存だけを救済する構造の説明が困難になる。厳格否定説では、省令10条書類は単なる補助証拠ではなく、みなし部分の申告上の外形形成に不可欠な法定添付資料 と位置づけるべきである。

5.4 「条約上の利益を下位規範で制限できない」論

日中条約などの tax sparing 条項は、日本の外国税額控除計算上、相手国で軽減・免除された税額を考慮に入れるよう求める。反対説は、下位規範である省令10条未添付を理由に、その条約効果を阻害することはできないと主張しやすい。

しかし、厳格否定説では、条約は 日本国の法令に従って 控除されることを予定しており、省令10条は条約利益の違法な切捨てではなく、日本の申告納税制度の中で、どの資料をもってみなし部分を主張立証させるか を定める国内手続法上の実装規定と読む。77_22はこの理解と整合的である。

6. 論証骨格と準備書面化の順序

6.1 最短の主張骨子

法人税法69条25項は、外国税額控除の適用について、申告書等への明細書の添付及び証拠書類の保存を要件として「場合に限り、適用する」と定めるから、これらは単なる訓示ではなく適用要件である。他方、同条28項は、保存がない場合に限って「やむを得ない事情」による救済を認めるにとどまり、添付欠缺について同様の救済を置いていない。実施特例法施行省令10条は、みなし外国税額控除に特有の計算明細書及び証明書類を、69条25項にいう添付書類の内容として具体化するものである。したがって、省令10条書類の欠缺は単なる立証不足ではなく、69条25項所定の適用要件の未充足である。そうすると、その欠缺を更正の請求段階で補完することは、通則法23条1項1号のいう「計算誤り」の是正ではなく、新たな制度適用要件の事後的充足又はみなし部分の適用範囲の追加的拡張に当たり、原則として許されない。

6.2 論証順序

  1. 69条25項の「場合に限り」を押さえ、添付と保存がともに適用要件であることを示す。
  2. 69条28項が保存だけを救済することを示し、添付欠缺に対する明文救済がない点を強調する。
  3. 省令10条がみなし部分の添付内容を具体化することを示し、25項との一体性を論じる。
  4. 23条1項1号は計算誤り又は法令違反を要求するにすぎず、新規追加型の制度適用には開かれていないと置く。
  5. 77_22と11812で、要件未充足の申告は適法であり、23条の入口に入らないことを確認する。
  6. 13531・13633で、現行法でも外形化されていない範囲の拡張が難しいことを補強する。
  7. 77_21と11242は、既に外形化された範囲内の誤り是正に限られると射程限定する。

6.3 裁判所向けに言い換えるなら

裁判所向けの書きぶりでは、「本件は書類の後提出が問題なのではなく、みなし部分の適用範囲が当初の申告書等に外形化されていなかったことが問題である」 と置くのが有効である。これにより、争点は「証拠の採否」から「申告納税方式における制度選択の成立時点」へ移る。

6.4 誠実に残すべきリスク

厳格否定説はかなり強く組めるが、無傷ではない。最大の弱点は、69条25項 と 省令10条 がともに 更正請求書 を明文で読む点である。したがって、主張は「更正請求書ルートが絶対に閉じる」とまで広げず、省令10条書類が欠けた結果、みなし部分の外形化が足りない場合には、23条1項1号に乗らない という形で絞る方が堅い。

7.3 実務上のメモ

  • 準備書面では、「後出しそのもの」 を非難するより、「外形化されていないみなし部分の新規追加」 と捉える方が強い。
  • 77_21・11242は避けずに先回りして扱い、当初から書類がそろっていた範囲内の再計算 という共通項で縛る。
  • 国税庁の当初申告要件廃止資料は、改正による入口拡張の存在を示す証拠としては有用だが、添付欠缺の後治癒を認める直接資料ではない。
  • 省令10条書類が欠ける事案では、みなし部分だけが落ちる成分別把握が重要であり、通常FTC部分と切り分けて論じる方が整合的である。
